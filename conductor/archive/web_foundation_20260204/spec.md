# Specification: MySealium Web Foundation

## 1. Goal

The objective of this track is to rebuild the current 7Sense placeholder into a high-performance, modern web foundation using Next.js. This serves as the technical baseline for the MySealium SaaS platform.

## 2. Requirements

- **Framework**: Next.js 15 (App Router).
- **Language**: TypeScript (Strict).
- **Styling**: Tailwind CSS for rapid, scalable design.
- **Motion**: Framer Motion for premium "Palantir-style" animations.
- **Responsive**: Mobile-first design following the "Modern Marine Utility" guidelines.

## 3. Design Tokens

- **Primary BG**: `#0a0a0f`
- **Secondary BG**: `#12121a`
- **Accent Cyan**: `#00d9ff`
- **Typography**: Inter (Variable font preferred).

## 4. Key Components

- **Navbar**: Secure, glassmorphic navigation.
- **Hero**: Pulsing orb visual with high-impact value proposition.
- **Feature Cards**: Data-driven cards with high-contrast utility.
- **Analytics**: Pre-configured for future event tracking.

## 5. Success Criteria

- [ ] Static export or server-side rendering works without errors.
- [ ] 100/100 Lighthouse performance score (or as high as possible with video/assets).
- [ ] Fully responsive down to 320px width.
